Query name/goal: Stars

Preview: Stars preview

SPARQL Endpoint: https://dbpedia.org/sparql

            SELECT DISTINCT ?subject ?subjectLabel ?subjectDescription ?property ?propertyLabel ?object ?objectLabel
            WHERE{
                ?subject a dbo:Constellation.
                ?subject dbp:family ?object.
                BIND(dbp:family as ?property).
                ?property rdfs:label ?propertyLabel. FILTER(lang(?propertyLabel)="en").
                OPTIONAL{?subject rdfs:label ?subjectLabel . FILTER(LANG(?subjectLabel )="en")}
                OPTIONAL{?subject rdfs:comment ?subjectDescription . FILTER(LANG(?subjectDescription )="en")}
                OPTIONAL{?object rdfs:label ?objectLabel. FILTER(LANG(?objectLabel)="en")}
            } LIMIT 100

Query name/goal: Similar museums

Preview: Similar museums preview

SPARQL Endpoint: https://dbpedia.org/sparql

            SELECT DISTINCT ?subject ?subjectLabel ?subjectDescription ?property ?propertyLabel ?object ?objectLabel
            WHERE{
                ?subject1 a dbo:Artwork.
                ?subject1 dbo:author ?property.
                ?subject2 a dbo:Artwork.
                ?subject2 dbo:author ?property.
                ?subject1 dbo:museum ?subject.
                ?subject2 dbo:museum ?object.
                FILTER(?subject!=?object)
                ?property rdfs:label ?propertyLabel.
                FILTER(lang(?propertyLabel)="en").
                OPTIONAL{?subject rdfs:label ?subjectLabel . FILTER(LANG(?subjectLabel )="en")}
                OPTIONAL{?subject rdfs:comment ?subjectDescription . FILTER(LANG(?subjectDescription )="en")}
                OPTIONAL{?object rdfs:label ?objectLabel. FILTER(LANG(?objectLabel)="en")}
            }

Query name/goal: Check if the story has a continuation...

Preview: Work and its continuation preview

SPARQL Endpoint: https://dbpedia.org/sparql

            SELECT DISTINCT ?subject ?subjectLabel ?subjectDescription ?property ?propertyLabel ?object ?objectLabel
            WHERE{
                ?subject a dbo:Work.
                ?subject dbo:subsequentWork ?object.
                BIND (dbo:subsequentWork AS ?property)
                dbo:subsequentWork rdfs:label ?propertyLabel.
                FILTER(lang(?propertyLabel)="en")
                OPTIONAL{?subject rdfs:label ?subjectLabel . FILTER(LANG(?subjectLabel )="en")}
                OPTIONAL{?subject rdfs:comment ?subjectDescription . FILTER(LANG(?subjectDescription )="en")}
                OPTIONAL{?object rdfs:label ?objectLabel. FILTER(LANG(?objectLabel)="en")}
            } LIMIT 100
