Query results must contain
subject,
subjectLabel - e.g., rdfs:label attached to subject in the English language. It will be visualize on the subject node in the metaverse.
subjectDescription - e.g., rdfs:comment attached to subject in the English language. It will be visualize as the node details in the metaverse.
predicate,
predicateLabel, - e.g., rdfs:label attached to predicate in the English language. It will be visualize on the edge in the metaverse.
object,
objectLabel - e.g., rdfs:label attached to object in the English language. It will be visualize on the object node in the metaverse.
